# 🎟️ EventoUni - Plataforma de Eventos da UniFSA

**Uma interface moderna e responsiva para gerenciamento de eventos universitários**, desenvolvida como projeto prático da disciplina **Programação Web** do Centro Universitário Santo Agostinho.

---

## ✨ O que foi desenvolvido?

- Header com menu responsivo  
- Banner hero com gradiente e imagem de fundo  
- Seção de filtros interativos  
- Grid de cards de eventos (totalmente flexível)  
- Footer com links de redes sociais  

Tudo construído usando **apenas HTML5 + CSS3 externo** com **Flexbox**, sem frameworks.

---

## 🛠️ Tecnologias utilizadas

- **HTML5** (estrutura semântica)
- **CSS3** (Flexbox, Box Model, Gradientes, Transições, Hover effects)
- Design responsivo (mobile-first)

---

## 🎯 Objetivo do projeto

Demonstrar o domínio dos conceitos aprendidos no **Guia Prático de CSS**:
- CSS Externo
- Flexbox (`display: flex`, `justify-content`, `align-items`, `gap`)
- Box Model
- Bordas, sombras, hover effects e transições
